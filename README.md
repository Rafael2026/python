## Reserverd words
The following words cannot be used as variables, constants or any other identifier names. All the Python keywords contain only lowercase letters.

| Header1 | Header2 | Header3 |
|---|---|---|
| and | exec | not |
| assert | finally | or |
| break | for | pass |
| class | from | print |
| continue | global | raise |
| def | if | return |
| del | import | try |
| elif | in | while |
| else | is | with |
| except | lambda | yield |
